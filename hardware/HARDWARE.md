## 各模块说明

### 1. DS18B20 — 桶内温度传感器

```
      DS18B20（防水探头）
      ┌─────────┐
      │         │
      │ 不锈钢管 │──── 三芯线 ──┬─ 红: VCC → 3V3
      │ 探头    │              ├─ 黄: DATA → GPIO4
      │         │              └─ 黑: GND → GND
      └─────────┘

      DATA 与 3V3 之间并联 4.7kΩ 上拉电阻
```

- **用途**：插入发酵液/酒液中，测量桶内核心温度
- **量程**：-55°C ~ +125°C，精度 ±0.5°C
- **通信**：OneWire 单总线，GPIO4
- **上拉电阻**：4.7kΩ（DATA → 3V3）**必须接**，否则通信失败
- **封装选择**：立创 EDA 中搜索 "DS18B20 TO-92" 或 "DS18B20 防水"

### 2. DHT22 — 环境温湿度传感器

```
      DHT22 模块（4 pin）
      ┌──────────┐
      │ DHT22    │
      │          │
      └──────────┘
       │  │  │  │
       V  D  N  N
       C  A  C  C
       C  T     (空)
       │  A  │
       │  │  │
      3V3 G5 GND
```

- **用途**：测量发酵环境（酒桶周围）的温湿度
- **量程**：温度 -40°C ~ 80°C，湿度 0~100%RH
- **通信**：单总线，GPIO5
- **注意**：购买模块版（已含 PCB 上拉电阻），无需外加电阻
- **封装选择**：立创 EDA 中搜索 "DHT22"，或用排针代替

### 3. 两路继电器模块 — 执行器控制

```
      继电器模组
      ┌────────────────────┐
      │  5V 两路光耦继电器  │
      │                    │
      │  VCC  ────  5V     │
      │  GND  ────  GND    │
      │  IN1  ────  GPIO16 │── 风扇
      │  IN2  ────  GPIO17 │── 加热
      │                    │
      │  COM1 ──── 火线     │
      │  NO1  ──── 风扇L    │
      │  COM2 ──── 火线     │
      │  NO2  ──── 加热器L  │
      └────────────────────┘
```

- **触发方式**：低电平触发（ESP32 GPIO 输出 HIGH 时继电器吸合）
- **高压侧**（市电 220V）：COM 接火线，NO 接负载，NC 悬空
- **注意**：`JD-VCC` 和 `VCC` 用跳线帽短接，使光耦工作在同一 5V
- **风扇规格**：DC 12V 涡轮风扇（通过继电器控制 220V → 12V 电源适配器通断）
- **加热器规格**：220V PTC 加热片或加热带（通过继电器控制通断）
- **封装选择**：立创 EDA 中用排针 + 文字标注即可

### 4. ESP32 供电

```
      MicroUSB 5V
          │
          ▼
      ┌──────────┐
      │ ESP32    │
      │ 开发板   │
      │          │
      │ 板载     │
      │ AMS1117  │── 3V3 输出 → 传感器供电
      │ 3.3V LDO │
      └──────────┘
```

- ESP32 开发板自带 MicroUSB 接口，直接用 5V 2A 充电器供电
- 板载 3.3V LDO 稳压器（AMS1117-3.3），输出 3V3 给传感器
- **不要**从 3V3 引脚给继电器供电（电流不足），继电器用 5V/VIN 引脚

## 连线检查清单

在嘉立创 EDA 中画图时，逐项核对：

- [ ] DS18B20 的 DATA 到 GPIO4 之间串联了 4.7kΩ 上拉电阻到 3V3
- [ ] DHT22 使用模块版本（内部已含上拉），DATA 直连 GPIO5
- [ ] 继电器 IN1 → GPIO16（风扇），IN2 → GPIO17（加热）
- [ ] 继电器 JD-VCC 和 VCC 短路帽已接
- [ ] 所有传感器 GND 共地
- [ ] DS18B20 和 DHT22 供电来自 3V3（不是 5V）
- [ ] 高压侧（继电器 COM/NO）标注"接市电 220V，注意安全"
- [ ] ESP32 通过 MicroUSB 5V 供电

## 嘉立创 EDA 绘制提示

### 搜索元件关键词

| 元件 | 立创 EDA 搜索词 | 推荐封装 |
|------|:--|------|
| ESP32 开发板 | `ESP32 DEVKIT` 或排针代替 | DIP-38 排母 |
| DS18B20 | `DS18B20` | TO-92 或 SIP-3 排针 |
| DHT22 | `DHT22` 或 `DHT11` | SIP-4 排针 |
| 4.7kΩ 电阻 | `4.7k` | 0805 或 AXIAL-0.3 |
| 2路继电器 | 排针代替 + 文字标注 | SIP-4 + 端子 |
| 风扇 | `FAN` 或电机符号 | — |
| 加热器 | 电阻符号 + 标注 | — |

### 绘制步骤

1. **新建原理图** → 放置 ESP32 符号（或 19×2 排针）
2. 放置 DS18B20（TO-92），DATA 脚串 4.7kΩ 接 VCC
3. 放置 DHT22（SIP-4），DATA 直连 GPIO5
4. 放置 2 路继电器模块（SIP-4 排针），IN1/IN2 接 GPIO16/17
5. 继电器输出端画开关符号，标注"风扇 220V"和"加热器 220V"
6. 放置 MicroUSB 供电端口
7. 检查 ERC（电气规则检查）无误

### 导出

- 原理图导出 PNG → 放入 `hardware/`
- 推荐也上传立创 EDA 源文件（.json 或 .epro）

## 安全警告

> ⚠️ **继电器控制 220V 市电时需要注意：**
> - 确保使用带光耦隔离的继电器模块
> - COM 端必须接火线（L），NO 接负载，零线（N）直连负载
> - 接线前务必断开电源
> - 继电器模块底部加绝缘垫片
> - 如需长期运行，建议加保险丝（5A）
> - 如不熟悉强电接线，请用电工协助
