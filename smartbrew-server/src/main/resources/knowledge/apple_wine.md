# 苹果酒/梅酒酿造知识

## 发酵原理

苹果酒（Cider）是利用酵母将苹果汁中的糖分转化为酒精和二氧化碳的过程。核心反应式：
**C6H12O6 → 2C2H5OH + 2CO2**

梅酒则分为两种工艺：浸泡法（白酒浸泡梅子）和发酵法（梅汁加糖发酵）。家庭酿造以浸泡法为主，发酵法类似苹果酒工艺。

## 温度范围

- **适宜温度**：15-25°C
- **最佳温度**：18-22°C
- **低于 10°C**：酵母休眠，发酵停滞
- **高于 30°C**：酵母死亡风险，产生杂醇油，酒体粗糙有异味

## 酵母选择

| 酵母类型 | 特点 | 适用场景 |
|----------|------|----------|
| 果酒专用酵母（如 EC-1118） | 发酵力强，耐受性好 | 新手推荐 |
| 野生酵母 | 风味复杂但不可控 | 有经验者尝试 |
| 啤酒酵母 | 酒精度低，风味独特 | 低度苹果酒 |

## 发酵阶段

1. **适应期**（0-24小时）：酵母活化繁殖，无明显气泡
2. **主发酵期**（3-10天）：剧烈冒泡，酒帽形成，糖度快速下降
3. **后发酵期**（1-3周）：气泡减缓，酒体澄清
4. **陈酿期**（1-6个月）：风味成熟，口感圆润

**总周期**：典型 2-4 周主发酵 + 1-6 个月陈酿

## 常见问题与解决方案

### 1. 发酵不启动（24小时后无气泡）
- **原因**：酵母失活、温度过低、果汁含防腐剂、糖度过高
- **解决**：检查温度是否在15°C以上；确认果汁无防腐剂（山梨酸钾等）；重新活化酵母后接种；测量糖度，若>25°Brix则稀释

### 2. 发酵突然变慢或停止
- **原因**：温度骤降、营养不足、酒精度过高抑制酵母
- **解决**：升温至20-22°C；添加酵母营养盐；检查是否已发酵完成（测糖度）

### 3. 酒液有异味（臭鸡蛋/硫味）
- **原因**：酵母缺氮，产生H2S
- **解决**：添加酵母营养盐；用铜质器具短暂接触（家庭不推荐）；充分换气（倒罐）

### 4. 表面出现白膜
- **原因**：酒花菌（产膜酵母）污染，接触氧气过多
- **解决**：轻微时可撇去白膜，加SO2，满桶密封；严重时需废弃

### 5. 酒液发霉（绿/黑/灰毛）
- **原因**：霉菌感染，卫生不到位
- **解决**：**必须废弃**，不可饮用。彻底消毒所有器具

## 关键参数

| 参数 | 理想范围 | 说明 |
|------|----------|------|
| pH值 | 3.3-3.8 | 过低抑制酵母，过高易染菌 |
| 初始糖度 | 12-20°Brix | 约可产生6-11%酒精度 |
| 终点糖度 | <1°Brix（干型） | 甜型需终止发酵后补糖 |
| 酒精度 | 5-8%（家庭苹果酒） | 可用糖调整目标酒精度 |

## 关键操作提示

- **消毒**：所有器具用食品级消毒剂（Star San等）处理
- **密封**：主发酵可用单向阀，陈酿必须满桶密封
- **倒罐**：主发酵结束后分离酒泥，减少苦味
- **装瓶**：确保发酵完全，否则有爆瓶风险
