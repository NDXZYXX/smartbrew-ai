# 葡萄酒酿造知识

## 发酵原理

葡萄酒酿造利用葡萄皮上的野生酵母或添加的纯种酵母，将葡萄汁中的葡萄糖和果糖转化为酒精：

**C6H12O6 → 2C2H5OH + 2CO2 + 热量**

红葡萄酒带皮发酵（萃取颜色和单宁），白葡萄酒去皮发酵（清汁发酵）。

## 酵母与添加剂

### 酵母类型

| 酵母 | 特点 | 适用 |
|------|------|------|
| EC-1118 | 发酵力强，耐受 18% 酒精度 | 通用/重启发酵 |
| RC-212 | 突出红色水果香气 | 红酒 |
| D47 | 突出白色花香，口感圆润 | 白葡萄酒 |
| 71B | 降酸能力，柔和口感 | 高酸葡萄 |

### SO2（二氧化硫/焦亚硫酸钾）

- **作用**：抗氧化、杀菌、抑制野生酵母
- **用量**：家庭酿造添加 50-100ppm（约 0.15-0.3g 焦亚硫酸钾/10L）
- **注意**：过量会产生刺鼻气味，且有害健康

## 温度控制

### 红葡萄酒
- **主发酵温度**：20-28°C（最佳 24-26°C）
- **温度过高（>30°C）**：酵母死亡，发酵停滞，产生异味
- **温度过低（<18°C）**：发酵缓慢，色素和单宁萃取不足
- **苹果酸-乳酸发酵（MLF）**：18-22°C

### 白葡萄酒
- **主发酵温度**：12-18°C（最佳 14-16°C）
- **低温发酵优势**：保留果香，口感清新
- **温度过低（<10°C）**：发酵可能停滞

## 关键步骤

### 1. 原料准备
- **葡萄选择**：成熟度好、无霉烂、糖度 20-24°Brix 为宜
- **清洗**：家庭酿造建议清洗后沥干（清洗会冲走野生酵母，需额外添加酵母）
- **去梗破碎**：红酒去梗后破碎（不破籽），白酒破碎后压榨取汁

### 2. 主发酵（5-14天）
- **红酒**：带皮发酵，每天压帽 2-3 次（将浮起的皮渣压回酒液中），促进色素和单宁萃取
- **白酒**：清汁发酵，直接接种酵母
- **判断终点**：气泡基本停止，皮渣下沉，糖度降至 0°Brix 左右

### 3. 压榨与分离（红酒特有）
主发酵结束后，用虹吸管或网袋分离"自流酒"，皮渣可轻压取"压榨酒"。压榨酒单宁更多，可单独陈酿或与自流酒混合。

### 4. 苹果酸-乳酸发酵（MLF，可选）
将尖锐的苹果酸转化为柔和的乳酸，降低酸度，增加复杂度。**红酒常做 MLF，白葡萄酒通常不做**（保留清爽感）。

### 5. 陈酿（3-12个月）
- **容器**：玻璃罐/不锈钢桶/橡木桶
- **要求**：满桶密封，避免接触空气
- **倒罐**：每 1-3 个月倒罐一次，除去酒泥
- **温度**：12-18°C，恒温避光

### 6. 装瓶
- 确保发酵完全（残糖 <0.2%）或已做灭菌处理
- 添加少量 SO2（30-50ppm）保护
- 使用深色瓶，避光保存

## 常见问题与解决方案

### 1. 臭鸡蛋味/硫味（H2S）
- **原因**：酵母缺乏氮源，产生硫化氢
- **解决**：添加酵母营养盐/磷酸氢二铵（DAP）；倒罐通气（氧化挥发性硫化物）
- **预防**：发酵初期补充酵母营养盐

### 2. 醋酸化（变成醋）
- **原因**：醋酸菌感染，酒液接触氧气过多
- **解决**：轻微时加 SO2 抑制；严重时无法挽救，只能当醋用
- **预防**：发酵容器密封良好（单向阀），陈酿时满桶

### 3. 发酵停滞（卡住了）
- **原因**：温度波动、营养不足、酒精度达酵母上限、残糖过低
- **解决**：
  - 升温至适宜范围
  - 添加酵母营养盐
  - 用 EC-1118 酵母重新起酵（活化后逐步加入）
  - 测量酒精度，若已达酵母耐受上限则接受现状

### 4. 氧化（颜色变棕、失去果香）
- **原因**：酒液接触空气过多，SO2 不足
- **解决**：立即加 SO2（50ppm），满桶密封
- **预防**：保持满桶，用惰性气体（氩气/氮气）保护顶空，定期检查 SO2 水平

### 5. 酒体浑浊不澄清
- **原因**：发酵未完全、蛋白质浑浊、胶体悬浮
- **解决**：
  - 耐心等待自然澄清（数周至数月）
  - 低温处理（放入冰箱 2-4°C，使酒石酸沉淀）
  - 使用澄清剂（膨润土、蛋清、明胶等）
  - 过滤后装瓶

## 关键参数

| 参数 | 理想范围 | 说明 |
|------|----------|------|
| 初始糖度 | 20-24°Brix | 约可产 11-13.5% 酒精度 |
| pH值 | 3.2-3.6 | 红酒偏高端，白葡萄酒偏低端 |
| 总酸（TA） | 5.5-8.5 g/L | 以酒石酸计 |
| 游离 SO2 | 25-35 ppm（陈酿） | 过低不足保护，过高影响口感和健康 |
| 酒精度 | 11-14%（干红/干白） | 取决于葡萄含糖量 |

## 家庭酿造安全提示

- **食品安全第一**：用具充分消毒，酒液无异味方可饮用
- **甲醇注意**：家庭自酿葡萄酒甲醇含量通常不高（远低于国家标准），但使用腐烂葡萄会显著增加甲醇
- **爆瓶风险**：装瓶前必须确保发酵完全，或使用耐压瓶（香槟瓶+铁丝扣）
- **适量饮用**：自酿酒酒精度不可控，饮用需谨慎
